## CRUD Biblioteca

## Pre-requisitos
Para a execução desse projeto localmente, você precisará instalar:
<ul>
  <li>Node.js (versão 14+)</li>
  <li>MySQL (instalado e configurado)</li>
  <li>Navegador Web (para ter acesso a interface</li>
</ul>
